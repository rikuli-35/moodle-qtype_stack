# 学生向け情報

STACKは、Moodle小テストで数学の問題を出題し、自動採点する機能を提供するソフトウェアである。

通常、解答はシンプルな線形構文を使って入力する。詳細は[解答入力](Answer_input.md)のページを参照できる。
入力した解答を_送信_すると、[評価](Answer_assessment.md)が行われる。

* STACKは、まず構文エラーの有無など、解答が_有効_であるかどうかを確認する。
* 次に、解答が_正解_かどうかを判定し、フィードバックを返す。
* 通常、問題には何度でも挑戦できる。
* ランダムに生成されたパラメータを含む問題が出題されることもある。この機能により、類題を繰り返し練習することもできる。

[アクセシビリティ機能](Accessibility.md)や、[等価推論](Equivalence_reasoning.md)入力の使い方についても参照できる。

また、よくある質問をまとめた[FAQ](FAQ.md)のページもある。
