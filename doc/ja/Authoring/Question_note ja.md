# 問題記録 (Question note)

問題記録は [CASText](CASText.md) です。問題記録は、ランダムに生成されたバリアントが同じか異なるかを決定するために使用されます。

2つの問題バリアントは、問題記録が等しい場合にのみ等しいとみなされます。

特に、学生の受験に関する統計を作成する場合、問題記録の等価性に従って受験をグループ化します。[問題変数](Variables.md#Question_variables) が異なっていても、2つのバリアントが必ずしも異なるとは限りません。したがって、問題記録は有用です。教師は、それぞれのユニークなバリアントを識別するための問題記録を選択する必要があります。これを自動化することはできません。

教師は、問題記録に答えに関する有用な情報を残すこともできます。
例えば、次のように問題記録を活用できます。

    \[ \frac{d}{d{@v@}}{@p@} = {@diff(p,v)@} \]

これは、特に学生が自分に与えられたバリアントについて質問する場合に非常に役立ちます。教師は問題記録を見るだけで、問題と答えの両方を得ることができます。

問題記録は、問題バリアントを [展開（デプロイ）](../STACK_question_admin/Deploying.md) するときに使用されます。
