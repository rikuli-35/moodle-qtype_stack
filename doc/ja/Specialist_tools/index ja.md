# 専門ツール

このセクションでは、「専門ツール」の使い方について詳しく説明する。このセクションの内容は、[クイックスタート](../AbInitio/index.md)やリファレンスドキュメント、特定の[数学トピック](../Topics/index.md)に関するアドバイスの範囲外に位置づけられる。

トピックには以下が含まれる。

* [ドラッグ＆ドロップ](../Specialist_tools/Drag_and_drop/index.md)問題タイプ（証明問題、グリッド問題、マッチング問題を含む）。
* [等価推論](../Specialist_tools/Equivalence_reasoning/index.md)：行ごとの推論の入力と評価のための機能。
* [JSXGraph](../Specialist_tools/JSXGraph/index.md)：インタラクティブなグラフィックス機能（解答欄としての使用方法を含む）。
* [GeoGebra](../Specialist_tools/GeoGebra/index.md)：インタラクティブなグラフィックス機能（解答欄としての使用方法を含む）。
* [グラフィックス](../Specialist_tools/Graphics.md)：グラフィックスを表示するための各種オプション。
* [順不同の複数解答欄](../Specialist_tools/Unsorted_multi_input.md)：独立した複数の解答欄を設定する方法。
