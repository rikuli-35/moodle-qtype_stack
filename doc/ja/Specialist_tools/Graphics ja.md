# STACKにおけるプロットとグラフィックス

プロットやグラフィックスは、問題テキスト、模範解答、フィードバックなど、あらゆる[CASテキスト](../Authoring/CASText.md)フィールドに配置できます。

STACKにプロット、画像、図表などを埋め込む方法はいくつかあります。

1. HTMLの `<img>` タグを使用して、外部画像を埋め込む。
2. [Maximaプロット](../CAS/Maxima_plot.md)を使用する。STACKの問題では `plot` コマンドを通じて `plot2d` の一部の機能を利用できます。
3. [JSXGraph](../Specialist_tools/JSXGraph/index.md)を使用して画像を作成する。
4. グラフ理論の[離散グラフ](../Topics/Discrete_mathematics.md)については、STACKの[`plot` コマンド](Plots.md)を使用し、離散プロットと線プロットを組み合わせることで直接作成できます。

注意事項：

* `draw` パッケージは現在サポートされていません。
* バージョン4.0以降では、`{#...#}` タグによってサードパーティのスクリプトと連携する機能が提供されています。この機能の使用例をお持ちの方は、開発者までご連絡ください。
